# Python整理

## 語法速查區

### Class類別

關鍵字 class 用來定義類別，類別是規劃物件object的藍圖，凡是物件的屬性attribute及方法method都要在類別中定義。

#### \_\_init\_\_\(\)

＿＿init＿＿\(\)為class預設的方法之一，其用途適用於建構construct 實體instance

白話文：一個正常人要有兩顆眼睛、兩只耳朵、一個鼻子、一個嘴巴，以class來寫即是：

```python
class Human():
    def __init__(self):
        self.
        self.
        self.
        self.
```
